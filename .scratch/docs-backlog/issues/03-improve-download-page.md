# Improve Download Page

Status: ready-for-agent
Priority: P1
Area: Download
Page: Download/download.mdx

## Problem

The download page is a primary user workflow but could do more to guide users to the correct installer and next step.

## Proposed Change

Add clearer installer selection guidance, release context, setup/upgrade links, and review whether verification details such as checksums are available.

## Acceptance Criteria

- Users can distinguish Desktop Windows, Desktop macOS, and VR Windows installers quickly.
- The page links directly to setup, upgrade, and requirements pages.
- Supporting documents are grouped clearly by release.
- Older releases remain accessible without distracting from the current release.
