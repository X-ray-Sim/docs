# Improve Welcome Page

Status: ready-for-agent
Priority: P2
Area: Welcome
Page: welcome.mdx

## Problem

The welcome page is highly visible but sparse. The support card has no body text, and the page does not orient different user roles.

## Proposed Change

Rewrite the intro description and cards so users can quickly choose the right path: VR setup, Desktop setup, downloads, support, and possibly procurement or IT review.

## Acceptance Criteria

- The page description is grammatically correct.
- Every card has useful body text.
- Card destinations match common user intents.
- The page remains compact and easy to scan.
